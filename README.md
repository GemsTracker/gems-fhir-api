# Gemstracker HL7 FHIR API
[HL7 Fhir](https://www.hl7.org/fhir/) resource server. Currently READ ONLY 

## Installation
```bash
 composer require gemstracker\gems-fhir-api
```

## Configuration
Add the ```\Gems\Api\Fhir\ConfigProvider::class``` to your config
